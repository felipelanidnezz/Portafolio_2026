Gemini API key required

This project uses Google's Gemini API to generate real, unique conversations between two AIs. You need a free API key to run it.

  1. Go to aistudio.google.com/apikey
  2. Click Create API key
  3. Open the file .env in the ouija-main folder
  4. Replace tu_api_key_aqui with your key
  5. Run npm start and reload this page
